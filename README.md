# IT Helpdesk Portfolio

This repository simulates real IT Helpdesk support tasks.

## Structure

- tickets: incident reports (user issues)
- runbooks: troubleshooting guides
- scripts: basic automation tools

## Tools Used in Scenarios

- ServiceNow – ticket logging and tracking
- TeamViewer – remote user support
- Active Directory – user account management
- Microsoft Entra ID – cloud identity and access

## Workflow

1. User issue is reported
2. Ticket is created in ServiceNow
3. Troubleshooting is performed
4. Remote support via TeamViewer if needed
5. Account checked in Active Directory / Entra ID
6. Issue is resolved
7. User confirmation is received
8. Documentation is updated

## Skills Demonstrated

- IT support troubleshooting
- Incident handling
- User account management
- Remote support
- Basic scripting
- Documentation

## Focus

All issues include verification to confirm resolution.
